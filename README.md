## Quick Setup

1. Install dependencies:
```bash
npm install
```

2. Add your Groq API key to `.env.local`:
```
GROQ_API_KEY=your_actual_groq_api_key_here
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000)

## Features

- **Savage Roast Modes**: Brutal, Gen-Z, or Gentle comedy feedback
- **Bias Filter Simulation**: See how different hiring managers perceive your resume
- **First Impression Score**: 3-second and 7-second scan analysis
- **File Upload**: Supports PDF, DOC, DOCX files

## Tech Stack

- Next.js 14
- TypeScript
- Groq API (Llama 3 8B model)
- PDF parsing
- File upload handling
